## 게임 소개

저승 명부(冥府), 업경대는 망자의 생전 삼독(貪·嗔·癡, 탐욕·분노·어리석음)을 비춘다.
당신은 하나의 혼(魂)으로 환생하여 십대왕이 다스리는 지옥을 층층이 내려가며,
흉물과 옥졸을 부적(符籍)과 무기로 물리치고 업(業)을 씻어 마침내 환생에 이르러야 한다.

한 번의 실패는 끝이 아니다. 왕을 격파할 때마다 새로운 혼과 힘이 해금되고,
다음 생(生)은 조금 더 멀리 나아간다.

## 게임 방법

- **목표**: 지옥을 층층이 내려가 각 지옥의 왕(보스)을 격파하고, 최종 환생에 도달한다.
- **조작**
  - 이동/공격: **방향키** 또는 **WASD** (적을 향해 이동하면 평타 공격)
  - 대기/상호작용: **Space** / **Enter**
  - 부적 사용·계단 하강 등: 화면 안내에 따라 선택
  - 모바일: 화면 하단 **터치 D-pad + 버튼**
- **핵심 시스템**
  - **혼(魂) 선택**: 떠돌이혼·무사혼·무당혼·동자혼·원귀혼 — 시작 방식이 달라진다 (왕 격파로 해금).
  - **부적(符籍)**: 화계·뇌계·치유·수면·순간이동 등, 슬롯에 담아 전투 중 사용.
  - **무기**: 환도·석장·염주·도깨비방망이 등, 저마다 다른 타격 규칙.
  - **업(業) 경제**: 처치·클리어로 업을 쌓아 다음 생의 강화·해금에 사용.
- **종료 조건**: HP가 0이 되면 그 생은 끝(사망) → 결과 화면 → 쌓인 업으로 다시 도전. 최종 왕까지 격파하면 **환생(클리어)**.

## 실행 방법

### 웹에서 바로 플레이 (심사자용)
위 **플레이 (웹)** 링크를 클릭하면 브라우저에서 즉시 실행됩니다. 설치·로그인 불필요.

### 로컬에서 실행
```bash
npm install
npm run dev      # http://localhost:5173 개발 서버
# 또는
npm run build    # dist/ 정적 빌드
npm run preview  # 빌드 결과 미리보기
```
요구 사항: Node.js 20+

## 기술 스택

- **언어/빌드**: TypeScript + [Vite](https://vitejs.dev/)
- **로그라이크 유틸**: [rot-js](https://ondras.github.io/rot.js/) (FOV·RNG 보조)
- **렌더링**: HTML5 Canvas 2D (도트 스프라이트 + 글리프 폴백)
- **배포**: GitHub Actions → GitHub Pages (`.github/workflows/deploy.yml`)

> 이 웹 클라이언트는 동일 게임의 **Godot 4 .NET(C#) 클라이언트**(Steam 출시 준비 버전)의
> 설계·콘텐츠를 참고하여 브라우저 플레이 가능 형태로 구현합니다.
